Installation

  1. Extract the contents of the archive and upload all the files to your web site directory.
  2. Change the permissions on the /albums/ and /cache/ directories to be writable by the server:
    chmod 777 albums cache
    or use your FTP program.
  3. Navigate to the setup url EXAMPLE: www.zenphoto.org/gallery/zp-core/setup.php, make sure everything checks out, and click go!
