Installation

  1. Extract the contents of the archive and upload all the files to your web directory.
  2. Edit the zp-core/zp-config.php.example file to meet your requirements and rename it as zp-config.php.
    • MAKE SURE YOU CHANGE YOUR ADMIN PASSWORD!!!!
  3. Change the permissions on the albums/ and cache/ directories to be writable by the server:
    chmod 777 albums/ cache/
    or use your FTP program.
  4. Navigate to the setup url Navigate to the setup url EXAMPLE: www.zenphoto.org/gallery/zp-core/setup.php, type in your credentials and you are finished!
